## 🚧 Projetos em destaque

### 🎵 Guess The Song
Minigame interativo onde o usuário escuta um trecho de música e tenta adivinhar.

**Stack:**
- React + Vite
- Node.js + Express
- PostgreSQL + Prisma

**Features:**
- Sistema de login (usuário/admin)
- CRUD de desafios
- Pontuação por dificuldade
- Futuro: integração com API do Spotify

### 🧙 Reino dos Magos
Sistema distribuído com comunicação via mensagens.

**Stack:**
- Java 17
- TCP Sockets
- JSON (Gson)

**Conceitos:**
- Arquitetura orientada a mensagens
- Separação COMMAND / EVENT
- NameService para descoberta de serviços
- Execução concorrente com threads
